# Agent Skills

This directory contains skills that extend the agent's capabilities.
Each skill should be a separate subdirectory containing a `SKILL.md` file.

## Structure
```
.agent/skills/
  └── my-skill/
      ├── SKILL.md       # Required: Instructions and metadata
      
## SKILL.md Format
The `SKILL.md` file must start with YAML frontmatter:
---

# Detailed Instructions
...
```
